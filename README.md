## Penjelasan
File ini adalah kumpulan nama file yang perlu diperhatikan dan dibaca terkait keperluan sistem ini.

## Manfaat dan Tujuan
File ini dibuat supaya membantu manusia dan AI dalam membaca dan melihat konsep aplikasi ini akan berjalan seperti apa.

## Kebutuhan Sistem
- Framework : Laravel 13
- Database : PostgreSQL 18

## List Catatan Kumpulan File
File berikut berada didalam path `*/catatan/`
### 1. ALERT INFORMATION PROMPT.md
Catatan yang berisi terkait apa yang tidak boleh dilakukan oleh pihak AI terhadap sistem disaat mengerjakan sistem ini.
### 2. DOKUMEN FITUR DAN HAK AKSES BERDASARKAN ROLE.md
Catatan yang berisi fitur dan hak akses masing masing role di dalam sistem aplikasi ini.
### 3. DOKUMEN FLOW SISTEM INVENTARIS & PERMINTAAN BARANG.md
Catatan yang berisi alur sistem ini secara kesuluruhan berdasarkan fitur dan role
### 4. DOKUMEN PROSEDUR OPERASIONAL SISTEM INVENTARIS & PERMINTAAN BARANG.md
Catatn yang berisi tentang prosedur yang dilakukan oleh user di dalam sistem saat penggunaannya nanti.
### 5. UNPRIORITY TASK
Unpriority Task adalah kumpulan task dan fitur yang belum perlu terlebih dahulu dilakukan guna menghemat penggunaan token, memaksimalkan pekerjaan lainnya dan memininalisir waktu pengerjaan.
